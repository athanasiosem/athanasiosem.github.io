Δημιουργεί όλων των ειδών τις εφαρμογές: απλές στατικές ιστοσελίδες, προχωρημένες Web εφαρμογές, APIs, γέφυρες, Desktop εφαρμογές, mobile εφαρμογές, κ.α.. Ειδικεύεται στις τεχνολογίες .NET, C#, SQL, HTML5, CSS3, JavaScript. Γνωρίζει πολλές τεχνολογίες ακόμα όπως Java, PHP, Node.Js, AngularJs, React κ.α..

Επικεντρώνεται στον καθαρό κώδικα, στην συντηρησιμότητα, στα best practices, στην ασφάλεια, στις επιδόσεις και στο SEO. Αγαπάει τον προγραμματισμό και επιδιώκει την ποιότητα στην δουλειά του. Ενημερώνεται καθημερινά για τα τελευταία νέα στον τομέα του Web & Software Development. Εξελίσσεται συνεχώς μαθαίνοντας νέες τεχνολογίες, νέες τεχνικές και best practices.
